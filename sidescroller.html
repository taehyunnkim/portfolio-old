First Development

As an avid gamer, creating a game was one of the things that I have always wanted to be able to do growing up. For that, I started learning C# and Unity engine to kickstart my journey as a PC/Mobile game developer. Learning C# was relatively simple: they both were object-oriented, and many of sytax in C# was very similar to that of Java.I wanted to make my first game as simple as possible; and for that, I was able to better acclimate myself to the new Unity environment.

This project really opened my eyes to the sheer volume of work that is put into game development. From programming to graphics design to animating, my respect for indie developers rose to a new level. Developing this first game was an incredibly joyful process, and I hope to bring many of my game ideas that rest in my notebook to fruition.
